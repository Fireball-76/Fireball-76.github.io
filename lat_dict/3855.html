aspectus

(adspectus), ūs m. [ aspicio ] (dat. aspectu = aspectui у V)
1) взгляд, взор
primo aspectu C — с первого взгляда
2) вид, зрелище
a. urbis C — вид города
pomorum jucundus a. C — приятный вид плодов
hominum aspectum vitare C — избегать вида людей
3) кругозор, поле зрения
orbes qui aspectum nostrum definiunt C — горизонт, ограничивающий наш кругозор
4) зрение
omnia, quae sub aspectum cadunt (veniunt) C — весь видимый мир
amittere aspectum C — потерять зрение, ослепнуть
5) появление, восход (siderum PM)
