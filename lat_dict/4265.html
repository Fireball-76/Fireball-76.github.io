auctio

ōnis f. [ augeo ]
1) увеличение, рост (tributorum T; dierum Macr)
2) публичные торги, аукцион
proponere (proscribere) auctionem C — объявить торги
auctionem proferre C — отложить аукцион
auctionem hereditariam constituere C — назначить распродажу наследства с торгов
3) продаваемое с торгов имущество (auctionem vendere C)
